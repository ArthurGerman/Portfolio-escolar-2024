Formulário de inscrição em concurso público

Essa atividade consiste num projeto referente ao módulo "Codificação para Back-End". Esse projeto tem como objetivo, desenvolver um formulário de inscrição em algum concurso público que, fação a aquisição dos dados do usuário em PHP, de forma que não ocorra nehum tipo de vazamento ou cálculo de dado para o front-end. Dessa maneira, possibilitando que todos os cálculos sejam executados no back-end e depois, retornados ao front-end através de tranferência de dados assim, garantindo mais segurança aos dados do usuário. Essa atividade foi muito importante para mim pois, através dela, pude adquirir conhecimentos importantes acerca de desenvolvimento back-end com PHP e o XAMPP. Abaixo estarei deixando um botão caso queira acessar o projeto no GitHub.
